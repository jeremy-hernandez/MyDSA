# Quick Guide

## Configuration files

- babel
- jest
- package.json

## Running main.js

- run: npm install
  - or: npm clean-install
- navigate to src/
- run: node main.js
  - or: npm start

## Testing

- run: npm test
- run: npm test:experimental
  - Useful flags (https://jestjs.io/docs/cli)
    - --testNamePattern=**pattern-name**: only test ran are those matching the pattern
