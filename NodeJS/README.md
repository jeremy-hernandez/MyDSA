# Quick Guide

## Configuration files

- babel
- jest
- package.json

## Run Project

1.  Install packages

    - npm install

      ```
      npm install
      ```

    - npm clean-install

      ```
      npm clean-install
      ```

2.  use npm start
    ```
    npm start
    ```

## Testing

- run: npm test
- run: npm test:experimental
  - Useful flags (https://jestjs.io/docs/cli)
    - --testNamePattern=**pattern-name**: only test ran are those matching the
      pattern
